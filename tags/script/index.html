Créer une Adresse mmmmail en Bash

{

Saturday 19 October 2013
article

Voilà mon script qui me permet de :

  • générer une adresse
  • la créer
  • l’ajouter à mon lecteur de flux

En dépendance, installez names avec pip : 

$ pip install names

et ça suffira. 

#!/bin/bash

URL="http://www.mmmmail.com"

# génère un nom grâce à 'names' ($ pip install names) 
mail="$(names | tr '[:upper:]' '[:lower:]' | tr '[ ]' '[\-]')-$(uuidgen | cut -b-3)"

wget $URL --post-data="mbox=$mail" --quiet > /dev/null

echo "<$mail@mmmmail.com> $URL/$mail.xml"

# pour les utilisateurs de newsbeuter (http://newsbeuter.org/)
echo "$URL/$mail \.mmmmail\"" >> ~/.newsbeuter/urls
